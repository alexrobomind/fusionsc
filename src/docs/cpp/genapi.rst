Generated Cap'n'Proto Struct Documentation
==========================================

This section contains the documentation for the Cap'n'Proto generated classes.
These classes are automatically generated from the Cap'n'Proto schema definitions
and provide type-safe data structures and RPC interfaces.

.. note::
   These are generated Cap'n'Proto struct and interface classes. For notes on
   how to deal with such classes, please consult the `Cap'n'Proto C++ examples
   <https://capnproto.org>`_ or the `Cap'n'Proto documentation
   <https://capnproto.org/encoding.html>`_.

For more information about the Cap'n'Proto schemas used in FusionSC, see the
:doc:`../capnp` section.

API Documentation
-----------------

The generated classes are organized by schema module. Each schema file produces:

- **Structs**: Data structures with typed fields
- **Interfaces**: RPC service definitions with methods and parameters
- **Clients/Servers**: C++ implementations for calling/implementation of services

For the complete list of generated classes, see the `Cap'n'Proto documentation index
<capnp/index.html>`_.
