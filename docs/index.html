Welcome to FusionSC’s documentation!

What is FusionSC?

Started as an effort to automate my personal workflow, FusionSC is a library intended to conveniently expose common operations required when analyzing the structure of magnetic confinement fusion devices, including the structure of the magnetic field as well as the machine geometry.
