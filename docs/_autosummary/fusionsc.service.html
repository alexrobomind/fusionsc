fusionsc.service

This module parses the Cap’n’proto schema files and exposes them as python classes

The following Cap’n’proto files are converted:

  • fusionsc.service: FusionSC specific service

  • fusionsc.service.capnp: Cap’n’proto builtins

  • fusionsc.service.devices: Device-specific service classes
